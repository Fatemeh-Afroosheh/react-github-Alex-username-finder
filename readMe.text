# GitHub User Search App

A simple React application that fetches GitHub users using the GitHub REST API and displays them in a stylish card layout.

## ✨ Features

- 🔍 Search for GitHub users with the keyword **"alex"**
- 👤 View each user's avatar and basic details
- ❌ Remove users from the list
- ⏳ Loading spinner while fetching data
- 🎨 Custom background and styling

## 🛠 Technologies Used

- React
- GitHub REST API
- JavaScript (ES6)
- HTML & CSS (custom)
